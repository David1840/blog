具体代码

Main

  1. 初始化
  2. 创建定时器，定时视频帧的刷新
  3. 创建解复用线程
  4. 等待事件

解复用

  1. 打开文件
  2. 找到音视频流
  3. 打开音频、视频流，创建视频解码线程，准备解码
  4. 读取packet，将音视频packet分别放入队列中，等待解码线程取出

视频解码

  1. 从队列中取出视频packet
  2. 解码，同步
  3. 加码后Frame存入数组，等待视频渲染
